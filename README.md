# IL2237_PCB_Design
The purpose of this project is to create an 4 layer PCB functioning as an ESP32 DevKit which can be used for a variety of applications. 

# Key Features 
- ESP32-C3 microcontroller
- USB-to-UART Bridge
- LiPo charger and battery management circuit
- Flash and SD card memory module
- Optical, audio and environment sensors

# Tool used
- KiCAD 9
